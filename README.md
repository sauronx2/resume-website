# Resume Website

Personal portfolio website for Oleksandr Serhadov - Senior QA Automation Engineer.

## Tech Stack

- Node.js + Express
- Tailwind CSS v3.4
- Vanilla JavaScript

## Quick Start

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Development with hot reload |
| `npm run build` | Build for production |
| `npm start` | Build and serve |

## Deployment

```bash
npm run build
npm run serve
```

Server runs at `http://localhost:3000`

## Structure

```
src/
├── index.html    # Main page (5 sections)
├── styles/       # Tailwind CSS
└── js/           # Navigation, animations
public/
└── assets/       # CV PDF, images
```
